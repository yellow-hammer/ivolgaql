# Вклад в ИволгаQL

Спасибо, что хотите помочь. Проект — **MVP** на Rust; придерживайтесь минимальных правок по смыслу задачи.

## Перед PR

1. В корне репозитория выполните `cargo build`.

2. Сообщения пользователю (поле `bird`, ошибки разбора, CLI, README) — **на русском**, коротко.

3. Правила для агента в Cursor: каталог [`.cursor/rules/`](.cursor/rules/).

## Оформление PR

Используйте [шаблон pull request](.github/pull_request_template.md). В описании — **что** сделано и **зачем**; при смене протокола или синтаксиса укажите это явно.

## Сообщения об ошибках

Оформляйте через [шаблон «Сбой»](.github/ISSUE_TEMPLATE/bug_report.md) или [«Предложение»](.github/ISSUE_TEMPLATE/feature_request.md): версия/ОС, шаги воспроизведения, ожидание и факт.
